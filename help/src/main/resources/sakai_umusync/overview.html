Sites sync tool: Overview

What it does

The sites synchronization tool umusync allows you to update permissions and tools.

Key concepts

Tasks In a sync task you must specify which sites you want to update, and which permissions/tools/pages these sites must have. How to create, edit or delete tasks.

Filters The filters are used to select those sites that have certain properties on the site. How to create, edit or delete filters.

Pages We can design pages with several tools to include them in sites later. How to create, edit or delete pages.

Who can execute a task

A user with admin role, or a Quartz job

  • Admin user:

    In the tool, the icon in the last columnn of the tasks list it's used to launch a task execution.

  • Quartz job:

    With the jobs scheduler based on Quartz we can plan the job Sites Sync Job, this job executes all the tasks in the list with active status (green row).
