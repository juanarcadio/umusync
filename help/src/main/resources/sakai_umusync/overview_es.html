Herramienta de sincronización de sitios: Introducción

Qué hace

La herramienta de sincronización de sitios umusync permite actualizar permisos y herramientas

Conceptos clave

Tareas En una tarea de sincronización se especifican que sitios se desea actualizar, y qué permisos/herramientas/páginas deben de tener esos sitios. Como crear, editar o borrar tareas.

Filtros Los filtros sirven para seleccionar aquellos sitios que tienen una serie de propiedades en el sitio. Como crear, editar o borrar filtros.

Páginas Podemos diseñar páginas con varias herramientas para posteriormente incluirlas en sitios. Como crear, editar o borrar páginas.

Quién puede ejecutar una tarea

Un usuario con rol administrador, o un trabajo de Quartz

  • Usuario administrador:

    Accediendo a la herramienta, el icono de la última columna de la lista de tareas sirve para lanzar la ejecución de un tarea.

  • Trabajo de Quartz:

    Con el planificador de trabajos basado en Quartz podemos planificar el trabajo Sync Sites Job, dicho trabajo ejecuta todas las tareas de lista que estén en estado activo (fila de color verde).
