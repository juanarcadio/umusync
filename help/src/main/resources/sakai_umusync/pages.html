How to Create/Edit/Delete pages

In the page that shows the pages list we can create, edit or delete pages.

To enter we have to click on the option Pages in the upper menu.

Create a new page

  1. Click on the option New Page
  2. Set a name to the page (it can't be the same as another page name, or a tool identifier name)
  3. Set the layout, the page can have its tools in one or two columns. If the layout was in two columns and you change to one, the tools on the left column will be added at the end.
  4. To add a new too to the page we click on the pull-down menu where all Sakai's tools appear, we choose the one we want to add and click on the green cross icon, which inserts that tool at the end of the last (or single) column.
  5. Once the tool is added, we can reorder them as we want dragging the name to another column, or another position in the same column.
  6. Once we fill in the page data we click on the button Save

Edit a page

The icon (pencil) in the first column of the pages list is used to edit them.

Notice that modifying pages can affect already created tasks using them.

Delete a page

The icon (red x) in the second column of the pages list is used to delete them.

Notice that deleting pages will also be deleted from those tasks using them.
